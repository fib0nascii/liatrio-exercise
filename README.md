 # Liatrio Exercise
 
### Prerequisites

````
  - Kubectl
  - AWSCLI
  - EKSCTL
  - Docker
````
The prerequisites must be installed and the cli configured before following these steps
### Usage

#### Clone repository
```git clone git@github.com:fib0nascii/liatrio-exercise.git```

#### Run deployment
```chmod +x ./deploy.sh```


```./deploy.sh```
